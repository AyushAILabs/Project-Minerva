# NLP Pipeline Project

## Overview

This project demonstrates a simple NLP pipeline using scraped text data from:

https://quotes.toscrape.com/

The workflow includes:
- web scraping
- corpus creation
- tokenization
- embeddings
- vector database storage

---

## Technologies Used

- Python
- BeautifulSoup
- NLTK
- SentenceTransformers
- ChromaDB
- Pandas
- NumPy

---

## Project Workflow

Scraping → Tokenization → Embeddings → Vector Storage
